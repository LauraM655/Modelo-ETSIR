Carga y Filtrado Automatizado de Datos (OpenDengue) de Sudamérica/Latinoamérica (Costa Rica, Rep. Dominicana, Colombia, Honduras, México) (PAHO)

Para asegurar la reproducibilidad del modelo y evitar fallos por conexiones de red externas, se realiza la lectura local del extracto temporal estandarizado del consorcio OpenDengue, aislando los registros semanales de interés.

# 1. Cargar librerías esenciales
library(readr)
library(dplyr)
library(lubridate)

# 2. Definir la ruta local exacta de tu archivo
archivo_local <- "Temporal_extract_V1_3.csv"

# 3. Validar existencia física del archivo
if (!file.exists(archivo_local)) {
  stop(paste("¡Error crítico! El archivo", archivo_local, "no se encuentra en el directorio."))
}

# 4. Leer la base de datos local
dengue_raw_ops <- read_csv(archivo_local, show_col_types = FALSE)

# 5. Definir los códigos ISO exactos de 3 letras
codigos_iso_interes <- c("CRI", "DOM", "COL", "HND", "MEX")

# 6. Pipeline de Limpieza Extrema de Espacios, Filtrado e Inserción de Filas
dengue_semanal <- dengue_raw_ops %>%
  # CRUCIAL: Eliminar espacios invisibles que tengan los textos a los lados en el CSV original
  mutate(
    ISO_A0_limpio = trimws(toupper(ISO_A0)),
    adm_0_name_limpio = trimws(adm_0_name)
  ) %>%
  # Filtrar usando la columna de códigos limpios sin espacios fantasmas
  filter(ISO_A0_limpio %in% codigos_iso_interes) %>%
  # Limitar la ventana temporal requerida (2014-2025)
  filter(Year >= 2014 & Year <= 2026) %>%
  # Crear la etiqueta del país basada en su código de 3 letras limpio
  mutate(
    pais = case_when(
      ISO_A0_limpio == "CRI" ~ "Costa Rica",
      ISO_A0_limpio == "DOM" ~ "República Dominicana",
      ISO_A0_limpio == "COL" ~ "Colombia",
      ISO_A0_limpio == "HND" ~ "Honduras",
      ISO_A0_limpio == "MEX" ~ "México"
    )
  ) %>%
  # Agrupar por país e ISO para colapsar todas las hileras municipales en conteos nacionales
  group_by(pais, ISO_A0_limpio, calendar_start_date, Year) %>%
  summarise(
    casos_totales = sum(dengue_total, na.rm = TRUE),
    .groups = "drop"
  ) %>%
  # Dar formato final y orden estructurado a las columnas para Laura
  transmute(
    pais = pais,
    codigo_iso = ISO_A0_limpio,
    fecha_semana = calendar_start_date,
    anio = as.integer(Year),
    # Calcular la semana epidemiológica basada en la fecha de inicio
    semana = as.integer(floor((calendar_start_date - make_date(Year, 1, 1)) / 7) + 1),
    casos_totales = casos_totales
  ) %>%
  arrange(pais, fecha_semana)

# 7. Imprimir resumen de verificación en la consola de RStudio
print(summary(dengue_semanal))
